About
A web application for people staying in
the same flat. Users can create and
join groups by entering a unique password
used when initially creating the group.
Our hallmark features include a shared calendar,
shared lists as well as a money tracker.
The shared calendar and list allows users
in the same group to view tasks or chores
as well as social events such as
nights out or birthdays.
The money tracker allows users from
the same group to track transactions
between each other as well as keep tabs
on how much money they owe each other.
